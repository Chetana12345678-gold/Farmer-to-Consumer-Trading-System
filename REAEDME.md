🌱 Farmer to Consumer Trading System

📌 Project Overview

The Farmer to Consumer Trading System is a web-based platform designed to connect farmers directly with consumers, eliminating intermediaries and ensuring fair pricing and transparency.

This system allows:

- Farmers to sell products directly
- Consumers to purchase fresh agricultural goods
- Admin to manage the system efficiently

---

🚀 Features

👨‍🌾 Farmer Module

- Register/Login
- Add, Update, Delete Products
- Manage Orders

🛒 Customer Module

- Register/Login
- Browse Products
- Add to Cart & Place Orders

🛠️ Admin Module

- Manage Users
- Manage Products
- Monitor Orders

---

🖥️ Technologies Used

- CSS
