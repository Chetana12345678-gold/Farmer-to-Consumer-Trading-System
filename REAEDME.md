# 🌾 Farmer To Consumer

A simple web application that connects farmers directly with consumers.  
This project is developed using **HTML and CSS** as part of academic learning.

---

## 📌 About The Project

Farmer To Consumer is designed to remove middlemen and allow direct interaction between farmers and buyers.  
It provides different sections like Home, Login, Registration, Products, and Administration.

This project demonstrates:

- Frontend web design
- Page navigation
- Layout structuring
- Basic UI styling

---

## 🚀 Features

- 🏠 Home Page with banner
- 🔐 User Login Page
- 📝 Customer Registration Page
- 👨‍🌾 Add Farmer Section
- 🛍 Add Product Section
- 📋 Farmer Details Page
- 🧾 Order Details Page
- 🎨 Clean and simple UI design

---

## 🛠 Built With

- HTML5
